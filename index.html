Art Break

Art Break is a way to use art to take a break from the world. A fully functional React app, Art Break receives live data from the Harvard Art Museums API, displays the data randomly, and allows you to draw on a digital sketchpad. *Cue deep breaths*

Feed Big Tex!

Feed Big Tex! is a matching game honoring my Texas roots with a State Fair of Texas theme. At the beginning of each round, a random number of visitors generates. The player must make matches to equal the number of fair visitors to win the game.

GitHub Repo Deployed Project

AllCity

AllCity is a full-stack React Native and Django mobile application. It is a platform for graffiti artists and lovers, alike, to share images of street art along with their location for others to enjoy. The brainchild of four engineers completely new to mobile development, AllCity is a labor of love and a work still in progress.
